## Description

`extarrows` (extensible arrows) provides some extensible arrows in math mode.

The list of new arrow commands:

* `\xlongequal`
* `\xLongleftarrow`
* `\xLongrightarrow`
* `\xLongleftrightarrow`
* `\xLeftrightarrow`
* `\xlongleftrightarrow`
* `\xleftrightarrow`
* `\xlongleftarrow`
* `\xlongrightarrow`
